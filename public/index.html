List Toys

Retrieve a list of toys with optional filtering and pagination.

Endpoint

GET /toys

Query Parameters

Parameter Description Route(example)
limit Number of toys to retrieve per page (default is 5). https://bored-goat-shorts.cyclic.cloud/toys?limit=1
page Page number (default is 1). https://bored-goat-shorts.cyclic.cloud/toys?page=1
sort Sort field (default is "_id"). https://bored-goat-shorts.cyclic.cloud/toys?sort=price
reverse Reverse sort order (set to "yes" to reverse, default is ascending). https://bored-goat-shorts.cyclic.cloud/toys?sort=price&reverse=yes
cat Filter by toy category. https://bored-goat-shorts.cyclic.cloud/toys?cat=educational
s Search for toys by name or info. https://bored-goat-shorts.cyclic.cloud/toys?s=plush

Endpoint

GET /toys/count

Query Parameters

Parameter Description Route
/count Returns the number of records in the collection https://bored-goat-shorts.cyclic.cloud/toys/price?min=10&max=40

Endpoint

GET /toys/price

Query Parameters

Parameter Description Route
/price?min&max Ability to receive products according to minimum and maximum price range https://bored-goat-shorts.cyclic.cloud/toys/price?min=10&max=40

Endpoint

GET /toys/single:id

Params Parameters

Parameter Description
/single:id Will pull only one record per ID

Example Request

GET /toys?limit=10&page=2&sort=name&reverse=yes&cat=educational&s=robot
