List Toys

Retrieve a list of toys with optional filtering and pagination.

Endpoint

GET /toys

Query Parameters

Parameter Description
limit Number of toys to retrieve per page (default is 5).
page Page number (default is 1).
sort Sort field (default is "_id").
reverse Reverse sort order (set to "yes" to reverse, default is ascending).
cat Filter by toy category.
s Search for toys by name or info.
min&max Ability to receive products according to minimum and maximum price range

Params Parameters

Parameter Description
/single:id Will pull only one record per ID

Example Request

GET /toys?limit=10&page=2&sort=name&reverse=yes&cat=educational&s=robot

Example Response

[
    {
        "_id": "1",
        "name": "Robo-Explorer Kit",
        "info": "This interactive kit allows kids to build and program their own robot...",
        "category": "Educational",
        "price": 49.99,
        "img_url": "https://example.com/images/robo-kit.jpg"
    },
    
]
